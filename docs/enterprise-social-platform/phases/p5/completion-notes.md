# P5 运营后台交接

## 已完成

- 新增 `/api/admin/**` 管理接口，管理员角色鉴权，首个用户自动补 `ROLE_ADMIN`。
- 后台支持频道、话题、话题别名、话题频道绑定、话题合并、内容审核、导入批次、导入项发布。
- 导入批次支持批量发布和软回滚，回滚不会物理删除帖子，只下架并拒绝审核。
- 前端新增 `/admin` 运营后台：总览、频道、话题、内容、导入五个工作区。
- 前端登录用户透传 `roles`，顶部导航和路由守卫支持管理员入口。

## 验证

- `backend`: `mvn test` 通过，5 个测试全部成功。
- `frontend`: `npm run build` 通过。

## 注意

- 已执行过 Liquibase 的数据库需要重新登录，让前端本地会话拿到新的 `roles`。
- 本阶段没有启动或保留任何长期运行的前后端服务。
